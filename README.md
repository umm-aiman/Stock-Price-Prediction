
# Stock Price Prediction using Machine Learning

## Objective
Predict the next day's stock closing price using historical stock market data.

## Dataset
Data retrieved from Yahoo Finance using the yfinance API.
## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn
- yfinance
## Models Applied
1. Linear Regression
2. Random Forest Regressor
## Features Used
- Open
- High
- Low
- Volume
## Results
- Random Forest achieved better prediction accuracy.
- The model was able to capture short-term stock trends effectively.
